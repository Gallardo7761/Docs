Introduction

SimpleTools is a small plugin that offers variety. It is not focused on any specific stuff. For example, it adds recipes, events, commands, chat format, etc.

By the way, I started it just when I started learning Java in university because our Programming Fundamentals professor told us that the better way of learning how to code is coding small projects yourself. Then time passed and the plugin has evolved to what it is now (although it still has a looooong way to go).

Commands

This is the list of all the plugin commands and their uses along with some GIF for better understanding.

  • /simpletools
    Aliases: /st

    Base/info command. It will show who developed SimpleTools (me 😛) but with cool format.

  • /streload
    Aliases: /str

    This will reload the whole config but stuff is obvious won't change even reloading (e.g. Inventory titles or Command descriptions)

  • /stconfig
    Aliases: /stc

    This will show an Inventory (or GUI) for you to enable/disable modules in-game (cool right?).

    Tip

    You don't need to reload after enabling/disabling modules!

  • /astick

    So this is a silly command that gives you a stick which will kill every entity with one touch. I did it for "debugging purposes" but I forgot to remove it so I called it *drumroll* The Admin Stick.

  • /discord

    This will send you your Discord invitation link if set in config.

  • /payxp <player> <levels>

    With this you can pay (or send) other players XP levels very easy. (And it has cool sounds btw)

  • /freefall <player>

    This was one of my firsts commands. It will teleport the target player to their Y position plus 200 blocks. (Useful for punishing in a funny way maybe?)

  • /globalchest [player]

    Ah yes, the Global Chest! It took me a long time since i was learning to code. 😭

    Anyway it is (as it name says) a chest that can be opened by every player so they can deposit and withdraw items from it. For example it could be used for community builds or somehitng like that.

    If you have the right permission you can open it for another player (generally this is for admins).

  • /thunder <player> <times>

    Ahhh the classic thunder strike... But this time you can decide how many thunders will strike the player.

  • /lobby [player]

    Sends you or another player to the lobby world if defined in config (you'll need the right permission to send another player).

  • /spawn [player]

    Sends you or another player to the spawnpoint of the world you/they are in (you'll need the right permission to send another player).

  • /sendcoords <player>

    With this you can send your coordinates to other players in a private message.

  • /wblock <world>

    This allows admins (or anyone with the right permission) to block certain worlds for players to not enter.

Permissions

This is the list of all the plugin permissions and their associated commands.

  • simpletools.*

    Allows you to use all the commands and access all the features of the plugin

  • simpletools.player

    Recommended permissions for players

  • simpletools.reload
    Commands: /streload, /str

    Allows you to reload the plugin configuration

  • simpletools.spawn
    Commands: /spawn

    Allows you to go to the spawnpoint via command

  • simpletools.spawn.others
    Commands: /spawn <player>

    Allows you to send other players to spawnpoint

  • simpletools.lobby
    Commands: /lobby

    Allows you to go to the lobby world via command

  • simpletools.lobby.others
    Commands: /lobby <player>

    Allows you to send other players to the lobby world

  • simpletools.freefall
    Commands: /freefall <player>

    Allows you to teleport players to the sky

  • simpletools.discord
    Commands: /discord

    Allows you to get the Discord invitation using the command

  • simpletools.thunder
    Commands: /thunder <player> <times>

    Allows you to strike a player n times with a thunder

  • simpletools.sendcoords
    Commands: /sendcoords <player>

    Allows you to send your coordinates to another player in a private message

  • simpletools.astick
    Commands: /astick

    Allows you to get the Admin Stick

  • simpletools.worldblock
    Commands: /wblock <world>

    Allows you to block a world

  • simpletools.chatformat

    Allows you to use text formatting in the chat (HEX is available)

  • simpletools.mentions

    Allows you to mention players

  • simpletools.adminchat

    Allows you to read and write in the admin chat

  • simpletools.globalchest
    Commands: /globalchest

    Allows you to open the Global Chest

  • simpletools.globalchest.others
    Commands: /globalchest <player>

    Allows you to open the Global Chest for another player

  • simpletools.globalchest.others
    Commands: /payxp <player> <levels>

    Allows you to pay/send XP levels to other players

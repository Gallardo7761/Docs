Introduction

Here you have all my projects well documented for you to understand every bit of them. :)

  • Contasoc

    So this is a Association Management Platform I made for "Huertos la Salud Bellavista", an Urban Garden Association of my neighborhood. It's been developed using Swing for the UI and Java (obviously) for the rest of it.

  • SimpleTools

    This is my first (and only for now) Spigot plugin. It is a variety plugin that adds some cool stuff and utilities too. It's goal is to be a big plugin useful for administrators and fun to use for players too.

  • Amodgus

    This is my first (and only for now) Forge mod. It adds Crewmates from Amongus to the game. You can pet them if you find them in the ships around the overworld sky and they'll help you to fight or explore.
